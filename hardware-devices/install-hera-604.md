# Install a Hera 604 router

Use this as a concise installation flow for a Hera 604 router. In a full migration, this page would be populated from the complete model-specific source article.

{% stepper %}
{% step %}
## Inspect the package

Confirm the router model, power supply, SIM, antennas, and any customer-specific configuration sheet.
{% endstep %}

{% step %}
## Insert the SIM

If a SIM is supplied separately, insert it fully and check the adapter fit before powering the router.
{% endstep %}

{% step %}
## Attach antennas

Attach cellular antennas to the LTE ports and attach Wi-Fi antennas if the deployment uses Wi-Fi.
{% endstep %}

{% step %}
## Power on and verify status

Power the router and confirm basic status indicators before mounting permanently.
{% endstep %}
{% endstepper %}
