# Configure basic settings

Eseye preconfigures the cellular connection for Hera routers, but field teams still need to validate settings and status.

## Settings to verify

<table>
<thead><tr><th>Area</th><th>What to check</th><th>Why it matters</th></tr></thead>
<tbody>
<tr><td>Router</td><td>Name, location, and local access details.</td><td>Helps support identify the device.</td></tr>
<tr><td>Mobile network</td><td>SIM details, registration state, signal, and APN.</td><td>Confirms the device can use the expected network.</td></tr>
<tr><td>Wi-Fi</td><td>Enabled state, SSID, and security settings.</td><td>Prevents local access confusion.</td></tr>
<tr><td>Status</td><td>System, cellular, and connection health.</td><td>Establishes a baseline before troubleshooting.</td></tr>
</tbody>
</table>

{% endhint %}
