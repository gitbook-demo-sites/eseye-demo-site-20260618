# AnyNet SIM hardware

Eseye supplies physical SIMs in different form factors, embedded SIMs, and iSIM options. Hardware selection should be made alongside device design, manufacturing flow, and operational support plans.

## Field checks

{% stepper %}
{% step %}
## Confirm the form factor

Check whether the device needs removable SIM, embedded SIM, or iSIM capability.
{% endstep %}

{% step %}
## Seat the SIM correctly

If using a 3FF or 4FF adapter, ensure the SIM is correctly seated so it does not catch or misalign in the device.
{% endstep %}

{% step %}
## Verify activation and status

Confirm the SIM is activated and visible in the platform before troubleshooting network behavior.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
A physical seating issue can look like a network problem. Always check SIM fit and device recognition before deeper connectivity diagnosis.
{% endhint %}
