# Hera router overview

Hera routers are preconfigured to customer specifications for easier deployment. Coupled with an AnyNet SIM, they support global connectivity for fixed wireless, edge, and enterprise IoT use cases.

## Common use cases

- Rapid field connectivity where wired broadband is unavailable or slow to provision.
- Backup connectivity for business-critical locations.
- Managed IoT deployments where router configuration should be repeatable.
- Remote monitoring and diagnostics across distributed sites.
