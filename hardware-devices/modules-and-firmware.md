# Modules and firmware

The current documentation includes module-related content, including guidance for devices such as Nordic nRF9160 and firmware or software update paths.

## Recommended documentation pattern

{% columns %}
{% column %}
### Device setup

- Manufacturer configuration prerequisites.
- SIM insertion and activation checks.
- Network registration expectations.
- Known modem command differences.
{% endcolumn %}

{% column %}
### Firmware operations

- Current approved firmware version.
- Upgrade prechecks.
- Rollback or recovery procedure.
- Support escalation path.
{% endcolumn %}
{% endcolumns %}

{% hint style="info" %}
Device and module pages are good candidates for GitBook reusable snippets, especially where SIM activation, APN settings, and support escalation steps repeat.
{% endhint %}
