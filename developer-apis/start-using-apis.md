# Start using APIs

Use this page to orient developers before they jump into individual API families.

{% stepper %}
{% step %}
## Confirm API access

Request credentials, confirm the portfolio or device scope, and identify whether the API is available for the customer account.
{% endstep %}

{% step %}
## Pick a test tool

Use an API development app such as Postman or an internal integration harness for early validation.
{% endstep %}

{% step %}
## Choose the first API family

Use [Choosing an API](choosing-an-api.md) to decide whether the task belongs to SIM, SMS, Tigrina, or another API group.
{% endstep %}

{% step %}
## Capture error behavior

Document returned error format, retry policy, and support evidence before moving to production automation.
{% endstep %}
{% endstepper %}
